Steps to run the application:

Step 1:Move to the directory by using the commmand,
  example:
  cd C:\Users\Vaibhav\Desktop\note_taking_app

Step 2:Create an environment by using the command:
  python -m venv myenv

Step 3: Activate the environment by command:
 myenv\Scripts\activate

Step 4: Install flask by:
  pip install flask

Step 5: Start the application by running following command:
  python app.py

Step 6: Open the following link in browser for output:
http://127.0.0.1:5000/
